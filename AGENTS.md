# 1) Что нужно сделать (цель и результат)

Нужно создать веб-приложение на Next.js, которое помогает тренировать учеников: оно **алгоритмически** генерирует задачи **ЕГЭ по математике (база), задание №9**, и для каждой сгенерированной задачи выводит:

1. **Условие задачи** (обычный текст на русском)
2. **Правильный ответ** (строка, **только число**, без единиц измерения)
3. **Пошаговое решение** (4 шага, как в образцах)

Генерация должна быть **строго программной**: параметры выбираются случайно (детерминированно по seed), ответ вычисляется формулой, решение строится из шагов с подстановкой чисел. Никаких “генераторов текста” в рантайме, кроме шаблонов.

Приложение должно уметь генерировать **три типа задач №9** (как в демоверсии):

* площадь части участка, не занятой домом;
* угол между соседними спицами;
* длина забора вокруг прямоугольника с проездом.

Пользователь управляет генерацией:

* выбирает **количество задач** (1–50),
* выбирает **тип** (один из трёх или “смешанный”),
* выбирает **сложность** (лёгкая/средняя/сложная),
* опционально задаёт **seed** (начальное число), чтобы получать одинаковый набор при повторе.

**Интерфейс полностью на русском.**
**Единицы измерения в ответе не писать.**
Все ошибки/уведомления показывать **только через Sonner** (снекбар).

### Адаптивный макет (Сетка + Sheet)

* **Телефон/планшет (< lg):** настройки прячутся в выдвижную панель **Sheet** по кнопке “Настройки”.
* **Компьютер (lg+):** слева постоянный блок настроек, справа результаты (две колонки).

# 2) Что должно быть на экране (поведение UI)

## Экран “Генератор”

### Блок управления (на русском)

* Количество задач (число)
* Тип задач (Смешанный / Площадь / Спицы / Забор)
* Сложность (Лёгкая / Средняя / Сложная)
* Начальное число (seed, необязательно)
* Кнопка “Сгенерировать”

### Результаты

Список карточек задач. В каждой карточке:

* заголовок: “Задание 9 — …”
* текст условия
* раскрывающийся блок “Ответ”
* раскрывающийся блок “Решение” (нумерованные шаги)

### Уведомления (только Sonner)

* “Сгенерировано: N”
* “Некорректное количество задач”
* “Не удалось сгенерировать набор. Измените настройки.”
* “Ответ скопирован” / “Не удалось скопировать” (если добавишь копирование)
